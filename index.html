Week 1: Basic HTML & CSS

HTML (Hyper Text Markup Language) is the most widely-used markup language in existence. It is used to semantically define the structure of a document. CSS, standing for Cascading Style Sheets, was created to allow developers the ability to style their pages with external files rather than using attributes in the HTML. Where HTML defines the content and structure, CSS provides the visual guidelines for the browser.
